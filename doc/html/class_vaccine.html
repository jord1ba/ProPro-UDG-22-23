◆ modifier()

abstract float Vaccine.modifier ( String  data,
float  value 
)
abstract

Retorna el valor value modificat segons la vacuna i el paràmetre de la vacuna que es vulgui saber mitjançant el paràmetre data.

Paràmetres
dataTipus de dada que s'està enviant per value.
valueValor de la dada previ a l'efecte de la vacuna.
Precondició
data és un dels següents valors:
  • virusEffectiveness
  • mortality
  • duration
  • fallSick
  • contagiousness
Retorna
Retorna el paràmetre value, que és del tipus especificat a data i, segons el tipus de vacuna, es retorna modificat o es retorna directament.

Reimplementat a SoftVaccine i HardVaccine.
