static void Writer.showMainMenu ( char  sum_or_log)
static

Mostra el menú principal.

Paràmetres
sum_or_log's' si s'està mostrant el resum o 'l' si s'està mostrant el log.
Precondició
tick >= 0 && last_tick >= 0 && tick <= last_tick.
Postcondició
Es mostra el menú principal corresponent a la situació i pàgina en què es troba l'usuari (amb diferents opcions segons el que l'usuari ha de poder fer).
Excepcions
IllegalStateExceptionsi sum_or_log no és ni 's' ni 'l'
