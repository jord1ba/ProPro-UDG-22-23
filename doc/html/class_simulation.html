void Simulation.setHarshLockdown ( Region  region,
boolean  state,
float  new_rate 
)

Estableix el confinament estricte de la regió a l'estat desitjat.

Precondició
new_rate >= 0 && new_rate <= 1 && region pertany a la llista de regions de la simulació.
Paràmetres
regionRegió que es vol confinar estrictament.
stateEstat del confinament.
new_rateNou ratio de moviments de dins de la regió.
Postcondició
Si state == true, region queda bloquejada amb el ratio intern de mobilitat new_rate. Altrament, region queda desbloquejada (si arestes de la regió estaven tancades es mantindran així).
