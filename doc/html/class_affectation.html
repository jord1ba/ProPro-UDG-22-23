static int Affectation.transfer ( Affectation  from,
Affectation  to,
int  people 
)
static

Transfereix el nombre de persones afectades people de l'afectació from a l'afectació to. Si to és null, s'esborren els grups de from i prou.

Precondició
l'afectació from ha de tenir almenys people persones que transferir.
Retorna
les transferències exitoses.

Això és una aproximació d'una distribució homogènia. El que busca és repartir de manera igualitària els moviments restants.
