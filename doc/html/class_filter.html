◆ Filter()

Filter.Filter ( Simulation  simulation)

Constructor principal que crea a partir de les llistes de regions i virus de simulation, les estructures que guarden Data.

Precondició
simulation té inicialitzats les estructures que guarden regions i virus.
Postcondició
_dataVR i _dataRV guarden Data de només aquelles combinacions de regió i virus que tenen alguna afectació, a més guarden les referències a les llistes de simulation i l'índex de l'últim virus.
Paràmetres
simulationSimulació d'on treure les regions i els virus.

◆ updateDataNextTick()

void Filter.updateDataNextTick ( )

Actualitza els objectes Data d'aquest objecte al nou tick generat, i afegeix el de les noves afectacions.

Precondició
Des de l'última crida d'aquesta funció, la simulació de la qual llegeix aquest Filtre ha generat un nou tick.
Postcondició
S'actualitzen tots els objectes de Data de _dataVR (i, per tant, de _dataRV), es llegeixen i carreguen els nous virus apareguts per mutació i es generen nous objectes Data per les afectacions aparegudes.
