Descripció Detallada

Classe encarregada de guardar les dades d'una afectació causada per un virus en una regió, o bé la suma d'un conjunt de Data (en aquest cas _virus i/o _region serien nulls).

Data.Data ( Region  region,
Virus  virus 
)

Constructor que rep un virus i una regió afectada per aquest i carrega i guarda les dades de l'afectació.

Precondició
region és afectada per virus. Cap dels dos és null.
Postcondició
Es carreguen totes les dades de l'afectació, inclosos els habitants i la possible vacuna.
Paràmetres
regionRegió afectada pel virus.
virusVirus que afecta la regió.

Iterator< Data >  iterator

Constructor que rep un iterator d'una collection de Data i realitza i guarda la suma dels valors de cada un dels objectes de la collection.

Precondició
True.
Postcondició
Aquest objecte guarda la suma de tots els objectes Data de la collection de iterator i si region i virus no són null, guarda la possible vacuna.
Paràmetres
regionLa regió que representa el conjunt de Data de iterator (pot ser null).
virusEl virus que representa el conjunt de Data de iterator (pot ser null).
iteratorIterator de l'estructura de la collection amb els objectes Data a sumar.
